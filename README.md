# MLKit — Classical ML Inference and Modeling Suite

**Course:** CS 771 - Machine Learning | IIT Kanpur
**Instructor:** Prof. Purushottam Kar
**Duration:** Jan 2026 - May 2026

## Overview

A collection of classical ML assignments implementing Bayesian inference, generative modeling, and kernel-based classification across real-world and benchmark datasets.

## Assignments

### Minor 1 — Linear SVC with Hyperparameter Tuning
- Optimized a **Linear SVC** via robust hyperparameter tuning (penalty, loss, C, tolerance)
- Achieved **100% accuracy** across diverse separability regimes on synthetic datasets

### Minor 2 — Bayesian Pipelines and MNIST Reconstruction
- Engineered **Bayesian pipelines** via Gaussian generative models
- Classified and reconstructed **51% masked MNIST images**

### Minor 3 — EM-trained Mixture Models
- Modeled **air pollution data** via EM-trained Gaussian Mixture Models
- Benchmarked rigorously against **Decision Tree Regressors**

## Tech Stack
`Python` `NumPy` `scikit-learn` `Matplotlib`
